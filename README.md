# YOUtility - Online Marketplace for Services

YOUtility is a comprehensive online marketplace that connects customers with trusted service providers for various home, auto, beauty, and other services.

## Features

### For Customers:
- **Browse Services**: Explore a wide range of services including electricians, plumbers, carpenters, mechanics, and more
- **Find Providers**: Search and filter service providers by location, rating, and availability
- **Book Services**: Easy booking process with detailed forms
- **Secure Payments**: Safe payment processing with multiple options
- **Track Bookings**: View booking history and manage upcoming appointments
- **Leave Reviews**: Rate and review completed services

### For Service Providers:
- **Create Profile**: Set up a professional profile showcasing skills and experience
- **Manage Availability**: Set working hours and availability
- **Receive Bookings**: Get notified of new booking requests
- **Build Reputation**: Earn reviews and ratings from satisfied customers

## Pages

1. **Splash Screen** (`index.html`) - Landing page with service overview
2. **Sign Up / Login** (`login.html`) - User authentication
3. **Home** (`home.html`) - Service categories and search
4. **Service Providers List** (`providers.html`) - Browse available providers
5. **Provider Profile** (`profile.html`) - Detailed provider information
6. **Booking Form** (`booking.html`) - Schedule service appointments
7. **Booking History** (`history.html`) - View past and upcoming bookings
8. **Payment Screen** (`payment.html`) - Secure payment processing
9. **Review Screen** (`review.html`) - Rate completed services

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Responsive design with modern styling
- **JavaScript (ES6+)**: Interactive functionality and form validation
- **Local Storage**: Client-side data persistence

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Navigate through the different pages using the menu
